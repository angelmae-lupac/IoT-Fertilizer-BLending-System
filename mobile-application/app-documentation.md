# Mobile Application Documentation

## Overview

The mobile application was developed using Kodular for the IoT-based fertilizer blending system. It provides real-time monitoring and remote interaction through Firebase Realtime Database (RTDB).

## Development Platform

- Kodular
- Firebase Realtime Database (RTDB)

## Features

### Real-Time Soil NPK Monitoring
- Displays nitrogen, phosphorus, and potassium readings from the NPK sensor.
- Provides real-time monitoring of soil nutrient values.

### Remote System Monitoring and Control
- Receives real-time data updates.
- Sends commands for system operation.

### Firebase Integration

Firebase Realtime Database is used as the communication layer between the mobile application and embedded system.

Data handled:
- NPK sensor readings
- System status
- Control commands
