# IoT-Controlled Fertilizer Blending Machine Based on Soil NPK Data

## Overview

An IoT-enabled automated fertilizer blending system that collects real-time soil NPK sensor data, synchronizes information through Firebase Realtime Database (RTDB), and enables remote monitoring and automated fertilizer dispensing through a mobile application.

## Features

- Real-time soil NPK monitoring
- Firebase Realtime Database integration
- Mobile application monitoring interface
- Remote machine control
- Automated fertilizer dispensing
- Nutrient requirement visualization
- Real-time data synchronization

## System Architecture

Sensor → Microcontroller → Firebase RTDB → Mobile Application → Control Command → Fertilizer Machine


## Technologies Used

- Embedded C/C++
- Arduino / Microcontroller
- Firebase Realtime Database
- Kodular Mobile Application
- IoT Communication
- Sensor Integration
- Real-time Data Management


## Project Components

### Embedded System

- Collects soil NPK sensor readings
- Processes sensor data
- Controls fertilizer dispensing valves
- Automates fertilizer mixing operation


### Mobile Application

- Displays real-time soil nutrient values
- Visualizes NPK requirements
- Sends control commands to the machine
- Provides monitoring interface
